# 期货量化策略监控系统 (Futures Quant)

这是一个基于 Python 的期货量化交易流水线，支持自动化行情同步、策略信号计算及钉钉预警推送。

## 🛠️ 核心功能
* **自动化流水线**：通过 `run_all.py` 一键执行全流程。
* **数据同步**：对接数据接口同步最新的期货历史行情。
* **信号监控**：基于 EMA 等指标生成每日交易指令清单。
* **自动备份**：执行前自动备份 `positions.json` 等关键配置文件。
* **钉钉推送**：支持将交易信号自动推送到指定的钉钉群聊。

## 🚀 快速开始
1. 确保已安装 Miniconda 环境并安装 `requests`, `pandas` 等依赖。
2. 准备 `config.json`（参考项目中的配置逻辑）。
3. 运行：
   ```bash
   python run_all.py
